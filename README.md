# ✈️ AI Travel Planner (Multi-Agent System)

An intelligent **AI-powered Travel Planner** that generates complete travel itineraries using a **multi-agent architecture**.
The system combines multiple specialized agents (flights, hotels, weather, attractions) to deliver a structured and realistic travel plan.

---

## 🚀 Live web app

 https://ai-travel-planner-ywaxuw3pq4vf983amugizy.streamlit.app/

---

## 🧠 Project Overview

This project demonstrates a **Multi-Agent AI System (Agent-to-Agent communication)** where:

* A **Root Agent** coordinates the workflow
* Specialized agents handle different tasks:

  * ✈️ Flight Agent
  * 🏨 Hotel Agent
  * 🌦 Weather Agent
  * 📍 Attraction Agent

All outputs are combined and enhanced using **Google Gemini LLM** to generate a **clean, day-wise itinerary**.

---

## ⚙️ Tech Stack

* **Language:** Python
* **LLM:** Google Gemini API
* **Frontend:** Streamlit
* **APIs Used:**

  * Geoapify (Places & Hotels)
  * Open-Meteo (Weather)
  * RapidAPI (Flights - optional/mock)
* **Architecture:** Multi-Agent System
